# Gemini PopClip Extension

独立仓库：Gemini PopClip 扩展。

## 功能

- 划词后点击 Gemini 图标
- 弹出输入框补充问题
- 自动拼接“原文 + 问题”并跳转 Gemini
- 兼容当前 Gemini Web URL 流程

## 安装

1. 打开 Releases 页面下载 `Gemini.popclipextz`
2. 双击安装到 PopClip

## 官方列表提交

- PopClip Extensions PR: https://github.com/pilotmoon/PopClip-Extensions/pull/1322
